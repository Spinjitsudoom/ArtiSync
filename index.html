Features

Everything your library needs

From fuzzy filename matching to embedded cover art — every step of the organisation workflow in one place.

🎵

Spotify Metadata

Search any artist and browse every release with album art thumbnails loading in the background. Pulls accurate track names, years, and genres directly from Spotify's database.

🔍

Fuzzy File Matching

Automatically maps your audio files to the correct tracks using Levenshtein edit-distance scoring. Works even with messy, numbered, or incomplete filenames.

🏷️

Tag Writing

Embeds artist, album, title, year, genre, track number, disc number, and cover art into ID3v2 (MP3), Vorbis comments (FLAC/OGG), and MP4/M4A atoms via TagLib.

🖼️

Cover Art

Downloads the highest-resolution image from Spotify's CDN and embeds it directly into your files. The sidebar shows 64×64 thumbnails for every release as they load asynchronously.

🔄

Audio Conversion

Convert between MP3, FLAC, M4A, OGG, and WAV via FFmpeg. AAC↔M4A is remuxed losslessly; all other pairs transcode with Low / Medium / High / Best quality presets.

↩️

Preview & Undo

The Preview tab shows every file → new name mapping with a confidence percentage before anything is changed. Every execute can be fully reversed with one click.

📁

Batch Mode

Process an entire artist root directory in one click — the app auto-detects and matches every album sub-folder without manual selection.

🎨

9 Colour Themes

Dark, Light, Midnight, Emerald, Amethyst, Crimson, Forest, Ocean, and Slate. Switch instantly from File → Settings with no restart required.

💾

Persistent Config

Your Spotify credentials, last-used directory, selected theme, and conversion settings are saved between sessions automatically.

Workflow

How it works

Six steps from a folder of messy audio files to a perfectly tagged library.

1

Browse to your music folder

Point the app at a root directory containing artist sub-folders. The path is remembered between sessions.

2

Artist auto-detected

The app searches Spotify for the folder name and populates the releases sidebar with thumbnails loading in the background.

3

Select a release

Click any album or single. The app fetches the full track list, cover art, and metadata from Spotify instantly.

4

Review the preview

The Preview tab shows every file → new name mapping with a confidence percentage. Green = high confidence, amber = uncertain.

5

Execute Rename + Tag

One click renames all files, writes every embedded tag, and embeds the full-resolution cover art.

6

Undo if needed

Changed your mind? Hit Undo to restore every original filename in one operation.

Installation

Get running in seconds

Two packaging formats available — pick whichever suits your system.

Requires qt6-qtbase and qt6-qtbase-gui on the host system.

On Fedora/RHEL:

sudo dnf install qt6-qtbase qt6-qtbase-gui

On Ubuntu/Debian:

sudo apt install qt6-base qt6-base-gui

Then download and run:

chmod +x MusicManager.AppImage
./MusicManager.AppImage
⬇ Download AppImage

Self-contained — no system Qt6 required. Works on any Linux with Flatpak installed.

flatpak install --user MusicManager.flatpak
flatpak run com.musicmanager.MusicManager
⬇ Download Flatpak

Requires CMake ≥ 3.16, a C++17 compiler, Qt6, and OpenSSL. libcurl and TagLib are auto-fetched if not found.

On Fedora:

sudo dnf install cmake gcc-c++ qt6-qtbase-devel openssl-devel

Build:

git clone https://github.com/Spinjitsudoom/MusicManagerUltimate
cd MusicManagerUltimate/"c++ source code"
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)
./build/MusicManager
Configuration

Spotify API setup

A free Spotify developer account is all you need — no paid subscription required.

1

Create a Spotify app

Go to developer.spotify.com/dashboard and click Create App. Name it anything you like.

2

Copy your credentials

From the app dashboard, copy your Client ID and Client Secret.

3

Paste into Settings

In Music Manager: open File → Settings, paste both values, and click Save.

4

You're connected

The status bar will show a green ● Spotify connected indicator. Start browsing your library.
